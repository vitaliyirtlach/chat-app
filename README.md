# Vue & NestJS chat application 
## Installation
```bash
npm install (in root folder)
cd server && npm install 
cd client && npm install 
```

## PostgreSQL database
```
RUN A POSTGRESQL SERVER
CREATE DATABASE vue-chat-app
```
