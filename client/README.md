## Description 

[Vue](https://github.com/vuejs/vue) is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run serve

# production mode
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test:unit
```

## Support
